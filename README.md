Currency Converter – Web Application

## Overview

This project is a simple currency converter built using HTML, CSS, and JavaScript. It allows users to convert values between currencies using predefined conversion rates.

The focus of this project is to understand how user input, calculations, and dynamic UI updates work together in a frontend application.

---

## Problem Statement

Performing quick currency conversions requires handling user input, applying conversion logic, and displaying results clearly. This project demonstrates how these elements can be combined in a structured web interface.
